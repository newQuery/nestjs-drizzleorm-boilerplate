# NestJs with Drizzle ORM

## Prerequiste

- Docker & Docker compose

## Installation

```bash
# build the environment
$ docker compose build

# install the dependencies
$ npm install

# start the development environment
$ docker compose up
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
